# 💸 Simulador de Parcelas

Um simulador de parcelamento moderno que calcula automaticamente o valor das parcelas com base no valor total e número de parcelas, aplicando juros dinâmicos de forma semelhante a sistemas reais de crédito.

## 🧠 Funcionalidades

* Inserção de valor da compra
* Definição do número de parcelas
* Cálculo automático em tempo real
* Aplicação de juros dinâmicos conforme quantidade de parcelas
* Exibição de:

  * Valor da parcela
  * Total pago
  * Total de juros
* Botão para copiar a simulação
* Feedback visual ao copiar (UX melhorada)

---

## 💰 Lógica de Juros

O sistema aplica taxas diferentes de acordo com o número de parcelas:

* Até 3x → 1.5% ao mês
* De 4x a 6x → 2% ao mês
* Acima de 6x → 2.5% ao mês

O cálculo é baseado em **juros compostos (Tabela Price)**, amplamente utilizado por instituições financeiras.
